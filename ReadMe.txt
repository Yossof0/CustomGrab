Follow These steps to download the extension.

1) Unzip and Copy The Extension folder to C:/Users/YOUR-SSD/Documents

2) Add your Preferable grab and grabbing cursor images NOTE: They have to be PNG and named as grab.png and grabbing.png, The size should be according to your cursor size

3) These Ones are Mac OS Grab and Grabbing cursors

4) Go to Manage Extensions in your Web Browser

5) Toggle On Developer Mode

6) Click Load Unpacked and browse in C:/Users/YOUR-SSD/Documents for the extension folder

7) Once done Open the test.html file

8) Hover over the orange box and you should be able to find your custom grab cursor

9) if you hold on the orange box you should see that custom grabbing cursor

10) Enjoy

NOTE: Never Move the folder from the original folder as this will result in the extension not working!
